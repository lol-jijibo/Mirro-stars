# Mirro-stars
Mirro 是一个面向年轻人（18-35岁）的 AI 驱动问题解决平台。用户输入自己的困惑或问题后，系统会自动完成：AI 分类 → 全网搜索真实案例 → LLM 生成结构化解答 → Mermaid 方案流程图 → 分步执行计划，全流程通过 SSE 流式推送实时呈现，给用户带来"AI 正在为你思考"的沉浸式体验。  与传统的 AI 问答不同，Mirro 不仅仅是"一问一答"，而是将每个问题当作一个需要被解决的项目来对待 —— 它会判断问题类型（行动型 vs 认知型），行动型问题会输出完整的流程图和执行步骤，认知型问题则提供多角度的深度分析。同时，每个答案都附有来自互联网的真实参考案例来源，增强答案的可信度
